This folder is for all 3D model files of the radio.

Files in this folder (R_models):

Any stl file prefixed with "Old..." in this folder is an old model file for only one face of 
the radio.

Main_box_assembly_part.SLDPRT
    File that was used to combine five different faces from the files prefixed with "Old..."

Main_box_assembly.SLDASM
    Edits on the Main_box_assembly_part.SLDPRT file.

radio_case_assembly.SLDASM
    Edits on the Main_box_assembly_part.SLDPRT file.

Main_box_assembly_part.STL
    The file that was used to print with the 3D printer.
